# OcladeAI

**OcladeAI** es una plataforma de inteligencia artificial conversacional, diseñada para ofrecer una experiencia premium, intuitiva y profesional. Inspirado en interfaces de élite, combina velocidad, estética y funcionalidad para resolver problemas, explorar ideas y mantener conversaciones inteligentes.

---

## ✨ Características

- 🧠 **IA Multimodal**: Lee imágenes, archivos y conversaciones.
- 🧩 **Selector de Modelos**: Cambia entre `Oclade Code`, `Oclade Think` y `Oclade Chat`.
- 📁 **Historial de Conversaciones**: Guarda y reanuda chats anteriores.
- 📷 **Subida de Archivos**: Imágenes, PDFs y documentos de texto.
- 🧮 **Contador de Tokens**: Visualiza tu uso en tiempo real.
- 🎨 **Diseño Profesional**: UI oscura con partículas animadas y tipografía moderna.
- 📱 **Responsive**: Optimizado para iPhone y PC.
- 😄 **Personalidad Humana**: Amable, graciosa y con emojis inteligentes.

---

## 🚀 Tecnologías Usadas

- **Frontend**: HTML5, CSS3, JavaScript ES6
- **Estilos**: Gradientes, animaciones CSS, tipografía moderna
- **UI/UX**: Partículas animadas (canvas), menú hamburguesa, tipografía fluida
- **Almacenamiento**: `localStorage` para historial
- **IA**: Integración con OpenRouter para acceso a múltiples modelos
- **Hosting**: Vercel
